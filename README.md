## About Eatable

this is an API that allows you to identify if the food is eatable or not.

## Installation
```
git clone git@github.com:chadidi/eatable.git
cd eatable
mv .env.example .env
composer update
php artisan key:generate
php artisan migrate
php artisan db:seed
```
